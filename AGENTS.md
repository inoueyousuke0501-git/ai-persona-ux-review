# AIペルソナUXレビュー

## プロジェクト概要
このプロジェクトは「AIペルソナUXレビュー」という、AIペルソナを利用したUX仮説検証支援ツール。

目的は、実ユーザー調査前の仮説生成とUI/UX改善速度向上。

## 開発方針
- MVPを優先
- 小さく作る
- まず動くものを作る
- 過度な抽象化を避ける
- UIデモ重視

## MVP機能
- 画像A/Bアップロード
- AIペルソナによる比較
- マス個人20人モード
- 結果サマリ表示
- ペルソナ別コメント表示

## 非MVP（UIのみ存在）
- 100人モード
- 1000人モード
- シニア層
- 富裕層
- 法人担当者

## 技術方針
Frontend:
- Next.js
- Tailwind
- Vercel

Backend:
- OpenAI API
- Supabase

## UX方針
このツールは「ユーザー調査の代替」ではなく、「仮説生成支援」として設計する。

## 禁止事項
- 実際の社内機密画像を利用しない
- 実在顧客データを保持しない
- AI結果を絶対的正解として扱わない
